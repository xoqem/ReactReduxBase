# ReactReduxBase
Boilerplate for a react / redux project.

## Dependencies

Install dependencies with yarn:

```
yarn install
```

## Development

### Manually building

Build the project:

```
yarn run build
```

Open dist/index.html in a browser.

### Watch

Have webpack watch for changes and automatically rebuild with:

```
yarn run watch
```

Open dist/index.html in a browser.
